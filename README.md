# D-Landing

A modern landing page project built with Next.js, React, TypeScript, and Tailwind CSS.

## Getting Started

### Prerequisites

- Node.js (v16+ recommended)
- npm

### Installation

```bash
npm install
```

### Development

```bash
npm run storybook
```

## Project Structure

- `src/components/` – React UI components
- `src/libs/` – Utility libraries (e.g., `cn.ts` for class name merging)
